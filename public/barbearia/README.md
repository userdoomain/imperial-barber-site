# Barbearia Imperial

Site estático institucional para a Barbearia Imperial — HTML5, CSS3 e JavaScript puro.

## Estrutura

```
/
├── index.html
├── css/style.css
├── js/script.js
├── images/
└── README.md
```

## Como usar

1. Baixe/descompacte a pasta.
2. Abra o `index.html` no navegador — pronto.
3. Para publicar: envie os arquivos para qualquer hospedagem (Hostinger, HostGator, cPanel, Netlify, GitHub Pages, Cloudflare Pages, Vercel).

Sem build, sem npm, sem dependências locais.

## Bibliotecas (via CDN)

- Google Fonts (Cinzel + Inter)
- Lucide Icons
- AOS (Animate On Scroll)

## Personalização

- Cores e tipografia: edite as variáveis CSS em `css/style.css` (`:root`).
- Textos e serviços: edite direto em `index.html`.
- Imagens: substitua os arquivos em `images/` mantendo os nomes.
- WhatsApp: altere o número nos links `wa.me/55...` no `index.html`.
